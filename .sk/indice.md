## 1. Visão Geral

Este projeto é um aplicativo React Native simples chamado **MeuAppNativo**.

Ele tem uma estrutura pequena, com:
- um ponto de entrada principal,
- um componente de cabeçalho,
- um componente que busca dados de uma API externa,
- arquivos de configuração básicos do app,
- e arquivos de apoio para build com EAS.

### Resumo funcional
O app atual:
1. abre uma tela principal,
2. mostra um cabeçalho com o nome do app,
3. faz uma requisição para uma API pública,
4. exibe os títulos recebidos.
