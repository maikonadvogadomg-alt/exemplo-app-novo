# MeuAppNativo

> Aplicativo nativo simples criado com React Native.

## Tecnologias
- React Native
- Axios para chamadas de API

## Setup
1. Clone o repositório.
2. Navegue até a pasta do projeto.
3. Execute `npm install` para instalar as dependências.
4. Para rodar o aplicativo no Android, execute `npm run android`.
